## 说明

Github角标

## 使用

```vue
<script setup lang="ts">
  import { GithubCorner } from "liyao-vue-common"
</script>


<template>
// 需要自己写css确定要放的位置， href为跳转的链接
 <GithubCorner :href="'https://github.com'"/> 
</template>
```



## 参数

```typescript
const props = defineProps({
  href: {
    type: String,
    default: '',
  }
})

```
