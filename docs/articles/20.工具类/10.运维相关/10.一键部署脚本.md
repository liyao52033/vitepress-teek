一键远程部署脚本，只需一行命令，即可开启您的Linux管理之旅

## 1、ssl

```sh
bash <(curl -fsSL https://xiaoying.org.cn/acme/safe.sh) https://xiaoying.org.cn/acme/acme.sh ssl
```

## 2、docker

```sh
bash <(curl -sL kejilion.sh) docker install
```

```sh
bash <(curl -sL kejilion.sh) frpc
```
